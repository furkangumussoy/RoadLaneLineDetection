# RoadLaneLineDetection

Detecting the lane lines using Python and OpenCV

## Dependencies
* Python 3.5
* NumPy
* OpenCV
* Matplotlib
* MoviePy

## How to run
To run the script stand-alone:

```
Usage: roadlaneline.py
```
